# Introdução ao Git e GitHub

Git é um sistema de controle de versão utilizado para gerenciar alterações no código.

GitHub é uma plataforma que hospeda repositórios Git na internet.

## Vantagens

- Controle de versões
- Histórico de alterações
- Trabalho em equipe
